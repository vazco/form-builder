# Troubleshooting

In this section, you will learn how to troubleshoot your application.
We're trying our best to resolve all issues described below.

## React Strict Mode

If you are using React Strict Mode, you can have an issue with Forminer. For example, the Form component doesn't register any input changes. This is a known issue in uniforms v3. Please disable React Strict Mode. It will be fixed when official uniforms v4 comes out.
